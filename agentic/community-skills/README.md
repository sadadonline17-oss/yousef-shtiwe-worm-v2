## Contributing

1. Create your `.md` skill file following the [Writing a Skill File](https://github.com/samugit83/yousef_shtiwe/wiki/Agent-Skills#writing-a-skill-file) format
2. Test it in your yousef_shtiwe instance by uploading via Global Settings
3. Fork the repo, add your `.md` file to this folder, and update this table with your GitHub username
4. Open a Pull Request
